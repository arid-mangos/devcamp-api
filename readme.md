# Devcamp API

> This is a backend api for a bootcamp sharing application based on brad traversy's API Mastercourse 

## Usage
Rename `config.copy.env` to `config.env` and add your settings.

## Install Dependencies
```
npm install
```

## Run
```
# dev mode
npm run dev

# prod mode
npm start
```
